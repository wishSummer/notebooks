## 二、AI + 后台系统架构设计（文字架构图）

```css
React 前端
  │
  │  REST / SSE
  ▼
Spring Boot 后端
  ├── Auth / RBAC
  ├── Config / CRUD
  ├── AI Service
  │     ├── Prompt 模板
  │     ├── 上下文拼接
  │     ├── 结果校验
  │     └── 降级 / 兜底
  │
  ├── Cache（AI 结果 / 业务数据）
  ├── DB（业务 + 审计）
  └── Vector DB（RAG）

```

### 核心原则

- AI 永不直写数据库

- AI 只给「建议」

- 所有 AI 输出可追溯
