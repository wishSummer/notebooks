# Vmware 笔记

## 虚拟机网络设置

1. 桥接模式：
   桥接模式允许虚拟机直接连接到物理网络，就像它是物理网络上的一个独立设备一样。这种模式适用于需要虚拟机与外部网络通信的场景。

    ```shell
    路由器   ->
            | -> 宿主机
            | -> 虚拟机
    ```

2. NAT 模式：
   NAT 模式允许虚拟机通过主机的网络连接访问外部网络，但虚拟机本身不会直接暴露在物理网络上。这种模式适用于需要虚拟机访问外部网络但不需要外部设备访问虚拟机的场景。

    ```shell
    路由器   ->
            | -> 宿主机 -> 虚拟机
    ```

3. Host-only 模式：
   Host-only 模式创建一个仅限主机和虚拟机之间通信的网络环境，虚拟机无法访问外部网络。这种模式适用于需要虚拟机与主机通信但不需要访问外部网络的场景。

    ```shell
    宿主机 <-> 虚拟机
    ```
